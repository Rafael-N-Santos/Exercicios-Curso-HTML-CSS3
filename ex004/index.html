Adicionando Favicon no HTML

Para adicionar favicom, utilizar o site favicon Generator
Logo apos do favicon feito, coloca-lo na pasta

Depois dentro do VScode
criar uma tag LINK:favicon na tag HEAD
e em 'href=' especificar o nome do arquivo entre parentênteses
